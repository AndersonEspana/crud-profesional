# CRUD Profesional de Usuarios

Sistema CRUD desarrollado en PHP y MySQL, enfocado en buenas prácticas
de backend, validaciones y diseño de base de datos.

Este proyecto forma parte de un portafolio orientado a
desarrollo backend y sistemas de información.

---

## Funcionalidades

- Crear, listar, editar y desactivar usuarios
- Validaciones de datos
- Uso de Soft Delete
- Manejo de roles
- Fechas de auditoría (creación / actualización)

---

## Tecnologías

- PHP 8
- MySQL
- HTML5
- JavaScript (Fetch / AJAX)
- XAMPP
